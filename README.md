## Setup 

### Tool Versions
<ul>
    <li>php 8.3.4</li>
    <li>nodejs 21.6.2</li>
    <li>postgres 14.1</li>
    <li>redis 7.0.10</li>
</ul>

### Run App
<ol>
  <li>
    Setup db configs in `.env`
  </li>
  <li>
    Run `php artisan db:create laravel_chatbot`
  </li>
  <li>
    Run `npm run dev` and `php artisan serve` on separate terminals
  </li>
</ol>
